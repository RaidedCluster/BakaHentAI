# BakaHentAI

Fine-tuning LLMs to learn Hentaigana (変体仮名) — the variant hiragana characters that were deprecated in 1900 but live on in Unicode (U+1B000–U+1B0FF).

## Structure

```
notebooks/          Training notebooks (Gemma 3 1B, Qwen 2.5 3B, Qwen 2.5 7B)
models/             LoRA adapters, tokenizers, and evaluation artifacts per model
```
